QSP / VEP / Stage299

Signature-Aware Verification Gate

Stage299 adds the Gate engine on top of the Stage288 public verification URL structure.

if signature_valid:
    decision = "accept"
elif verification_ok:
    decision = "pending"
else:
    decision = "reject"

accept

Evidence is valid and signature is valid.

pending

Evidence is valid, but signature is missing or not yet verified.

reject

Evidence is invalid. Fail-closed Gate rejects it.
